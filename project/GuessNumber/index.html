猜数字

游戏简介

本游戏通过讯飞云平台提供的Android SDK来实现语音输入。应用处理用户的语音输入以完成游戏过程。游戏提供两种模式：聚会模式与经典模式。聚会模式为聚会中常玩的猜数字游戏，而经典模式则为经典的费脑的猜数字模式。

游戏玩法

  1. 选择相应的模式，进行游戏。
  2. 通过“来一局”可以创建新游戏。
  3. 通过“结束”可以结束当前游戏。
  4. 通过“猜一下”进行语言输入。
